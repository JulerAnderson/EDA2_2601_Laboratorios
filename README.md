# Laboratorios de Estructura de Datos II – 2026-1

¡Bienvenidos al ciclo 2026-1!

Este repositorio servirá para compartir los laboratorios, ejercicios y materiales prácticos que trabajaremos a lo largo del curso de **Estructura de Datos II**.

## Objetivo del repositorio

Este espacio tiene como propósito:

- compartir ejercicios para practicar los temas vistos en clase;
- organizar los laboratorios y materiales por sesiones o temas;
- reforzar los conceptos del curso mediante problemas aplicados;
- brindar una base de apoyo para el repaso y la preparación de prácticas o evaluaciones.

## ¿Qué encontrarán aquí?

A lo largo del ciclo se irán publicando ejercicios y materiales relacionados con temas del curso.
